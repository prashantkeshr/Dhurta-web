Privacy Policy

Effective: 15 July 2026  ·  Dhurta.inc  ·  v1.0

Short version: Dhurta does not collect, transmit, or sell any personal data. Everything the browser stores stays on your device. You can inspect, export, or delete it at any time from the Transparency Dashboard.

1. Who We Are

Dhurta is a desktop browser application developed and maintained by Dhurta.inc. Contact: blackscont00@gmail.com.

2. What Data the Browser Stores — and Where

Dhurta stores all user data locally in a SQLite database on your own device. No data is sent to Dhurta servers. The following information may be stored locally:

  • Browsing history (pages you visit)
  • Bookmarks you create
  • Downloaded file records
  • Extension configurations
  • Security and feature toggle settings (e.g. VPN on/off, Anti-Fingerprint state)
  • Speed Dial tiles

You can view every stored value at any time by opening the Transparency Dashboard (eye icon in the sidebar) and export the full dataset as JSON.

3. What We Do NOT Collect

  • No analytics, crash reports, or telemetry are transmitted by default
  • No account or email address is required to use Dhurta
  • No browsing behaviour, search queries, or page content is sent to Dhurta.inc
  • No advertising identifiers or fingerprinting tokens are generated

4. Ghost Mode (Tor)

When Ghost Mode is active, browsing traffic is routed through the Tor network using a bundled Tor binary. Ghost Mode sessions are in-memory only — no history, cookies, or session data is written to disk. Dhurta does not operate Tor exit nodes and has no visibility into Ghost Mode traffic.

Tor is developed and maintained by the Tor Project, a non-profit organisation. Refer to the Tor Project's own documentation for information about Tor's privacy properties.

5. Built-in VPN

The built-in VPN feature connects to a third-party VPN service. When the VPN is active, your internet traffic passes through that provider's servers. Dhurta does not log VPN connection metadata. However, the VPN provider's own privacy policy governs what they may log. We recommend reviewing it before enabling the VPN for sensitive activity.

6. GitHub Releases

Dhurta is distributed through GitHub Releases (github.com/prashantkeshr/Dhurta). Downloading the application is subject to GitHub's Privacy Policy. The Dhurta website fetches release metadata from the GitHub public API to display live download links — this request is made by your browser directly to GitHub and is not proxied by Dhurta.inc.

7. This Website (dhurta.in)

This marketing website is a static site hosted on GitHub Pages. It does not set cookies, does not use analytics, and does not collect form submissions. GitHub may log standard server access logs (IP address, user-agent, timestamp) as part of hosting. Refer to GitHub's Privacy Statement for details.

8. Children's Privacy

Dhurta is not directed at children under 13 years of age. We do not knowingly collect personal information from children.

9. Your Rights

Because Dhurta stores no personal data on its own servers, there is no account to delete and no profile to request. All data is on your device and fully under your control. You can:

  • View all stored data via the Transparency Dashboard
  • Export your data as JSON
  • Wipe all stored data using the Nuclear Wipe function (sidebar)
  • Uninstall Dhurta to remove the application and its data directory entirely

10. Jurisdiction

Dhurta.inc operates under the laws of India. This policy is governed by and construed in accordance with Indian law.

11. Changes to This Policy

We may update this Privacy Policy to reflect changes in the application or legal requirements. The effective date at the top of this page will be updated accordingly. Continued use of Dhurta after any changes constitutes acceptance of the revised policy.

12. Contact

For any privacy-related questions: blackscont00@gmail.com
